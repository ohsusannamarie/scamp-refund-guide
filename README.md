## About This Project

Summer Camp 2026 was cancelled on May 8, 2026. Following the cancellation, many attendees were left navigating refund requests, payment disputes, and varying guidance from ticketing providers and financial institutions.

This guide was created to consolidate practical information into a single resource and help attendees understand available options regardless of how they paid.

The guide is maintained as information evolves and new developments become available.
